# Metarello

## Get started with metaUI
> [metaui.io](https://metaui.io)

### Commands

```bash
# install metaui
ng add @ngx-metaui/rules

# create a rule set for a model
ng g  @ngx-metaui/rules:mPage --modelClass=User --name=UserDetail 

# compile rules
npm run compile:oss

# start your angular app
npm start
```
